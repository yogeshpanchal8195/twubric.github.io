## Details
Twubric data can be sorted according to given fields (total, friends, influence & chirpiness) by clicking on them or by using shortcuts by pressing keys 1,2,3,4 respectively.
A paticular twubric detail can be seen by clicking on that particular twubric. 
Twubric data can be filtered according to the dates by inputting the start date & end date.
Twubric data can be deleted by pressing the cross button for every twubric.
